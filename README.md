# Quixmath
Simple yet robust CLI calculator tool.

## Description
This utility program was made with the intent of not needing to rely on
python's IDLE, serving as a quicker alternative to it.
(I know `bc` and already exists but this is intended to be simpler).

## Building
This project is built using `make`.

## Usage
`$ qm [option] <"equation">`.

### Dependencies
- GNU MPFR: for arbitrary-precision arithmetic;
